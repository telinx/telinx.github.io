聊聊Spring——常用Util工具类

发表于 2017-04-25 | 分类于 Spring

　>开发很多项目，都需要使用到Spring 这个框架。我们时常会写一些工具类，然后需写代码测试自己所写的工具方法，验证是否可靠。对于有看spring 源码习惯的人，知道spring框架本身自带了很多工具类，具体信息请参考spring手册。
　>

内置的resouce类型

UrlResource

工具类 描述
UrlResource 通过java.net.URL来访问资源，当然它也支持File格式
ClassPathResource 加载资源文件 new ClassPathResource(“resource/ApplicationContext.xml”);
FileSystemResource 通过 FileSystemResource 以文件系统绝对路径的方式进行访问
ServletContextResource 通过 ServletContextResource 以相对于Web应用根目录的方式进行访问
InputStreamResource 是针对于输入流封装的资源，它的构建需要一个输入流
ByteArrayResource 是针对于字节数组封装的资源，它的构建需要一个字节数组
EncodedResource
VfsResource 代表Jboss 虚拟文件系统资源
ResourceUtils 用于处理表达资源字符串前缀描述资源的工具.

工具类

工具类 描述
AnnotationUtils 处理注解
PathMatchingResourcePatternResolver 用 于处理 ant 匹配风格，找出所有的资源, 结合上面的resource的概念一起使用,对于遍历文件很有用
PropertiesLoaderUtils 加载Properties资源工具类,和Resource结合
BridgeMethodResolver 桥接方法分析器
GenericTypeResolver 范型分析器, 在用于对范型方法, 参数分析.
NestedExceptionUtils 待补充

xml工具

工具类 描述
AbstractStaxContentHandler
AbstractStaxXMLReader
AbstractXMLReader
AbstractXMLStreamReader
DomUtils
SimpleNamespaceContext
SimpleSaxErrorHandler
SimpleTransformErrorListener
StaxUtils
TransformerUtils 待补充

web相关的工具

工具类 描述
CookieGenerator 要创建Cookie
HtmlCharacterEntityDecoder
HtmlCharacterEntityReferences
HtmlUtils 转换为HTML转义字符表示、 转换为数据转义表示 转换为十六进制数据转义表示
HttpUrlTemplate 用字符串模板构建url, 它会自动处理url里的汉字及其它相关的编码. 在读取别人提供的url资源时, 应该经常用
JavaScriptUtils
Log4jConfigListener 用listener的方式来配制log4j在web环境下的初始化
UriTemplate
UriUtils 处理uri里特殊字符的编码
WebUtils 获取 HttpSession 特定属性名的对象 给session中的指定属性设置值 将 Map 元素添加到 ServletRequest 的属性列表中 获取 HttpServletRequest 中特定名字的 Cookie 对象
WebApplicationContextUtils 获取WebApplicationContext

其它工具集

工具类 描述
AntPathMatcherant 风格的处理
AntPathStringMatcher
Assert 断言,在我们的参数判断时应该经常用
CachingMapDecorator
CollectionUtils 用于Class的处理
ClassUtils ClassUtils.getShortName() //获取短类名、 ClassUtils.getClassFileName() //获取类文件名、ClassUtils.getPackageName() //获取包、 ClassUtils.getQualifiedName()
CommonsLogWriter
CompositeIterator
ConcurrencyThrottleSupport
DefaultPropertiesPersister
DigestUtils 摘要处理, 这里有用于md5处理信息的
FileCopyUtils 文件的拷贝处理, 结合Resource的概念一起来处理,
FileSystemUtils 递归复制、删除一个目录
LinkedCaseInsensitiveMapkey 值不区分大小写的LinkedMap
LinkedMultiValueMap 一个key可以存放多个值的LinkedMap
Log4jConfigurer 一个log4j的启动加载指定配制文件的工具类
NumberUtils 处理数字的工具类, 有parseNumber 可以把字符串处理成我们指定的数字格式, 还支持format格式, convertNumberToTargetClass 可以实现Number类型的转化
ObjectUtils 有很多处理null object的方法. 如nullSafeHashCode, nullSafeEquals, isArray, containsElement, addObjectToArray, 等有用的方法
PatternMatchUtilsspring 里用于处理简单的匹配. 如 Spring’s typical “xxx”, “xxx” and “xxx” pattern styles
PropertyPlaceholderHelper 用于处理占位符的替换
ReflectionUtils 反映常用工具方法. 有 findField, setField, getField, findMethod, invokeMethod等有用的方法
SerializationUtils 序列化与反序列化. serialize与deserialize方法
StopWatch 一个很好的用于记录执行时间的工具类, 且可以用于任务分阶段的测试时间. 最后支持一个很好看的打印格式. 这个类应该经常用
StringUtils 字符串常用工具类
SystemPropertyUtils
TypeUtils 用于类型相容的判断. isAssignable
WeakReferenceMonitor 弱引用的监控
