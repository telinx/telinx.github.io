聊聊Java——秒杀系统

发表于 2017-05-12 | 分类于 Java提高

本文讲述了关于秒杀系统的一种开发策略，上文通过序列图更加清晰的了解，下文主要讲述了代码的实现。

序列图：

系统初始化模块，系统启动

登陆秒杀初始页面，当前session生成验证码【/verifyCode，返回验证码】

用户提交秒杀产品ID和验证码，后台验证合法性，返回跳转路径【/path，返回path】

用户浏览器拿到路径path，提交信息给产品URL，【/{path}/do_miaosha】

接收到消息

排队时页面定时查询是否秒杀成功【/result】

//1.查询数据库中是否有该用户该产品的秒杀订单，有则返回秒杀成功，提示秒杀成功，跳转支付页面
//2.查询队列中是否有该用户的秒杀消息，有则返回前面还有多少人在排队中
//3.查询数据库该产品的存量，存量为0则提示已经被秒杀
​
